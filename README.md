Descripcin del sistema: Es un programa que evalua contenidos para decidir si pueden publicarse en una plataforma de streaming. por medio de los datos de contenido, duracion, horario, 
clasificacion y produccion, y da un resultado automático (entre: publicar, rechazar o revisar).

Instrucciones para ejecutarlo: Para usar el programa, primero se abre en la computadora con un editor de codigo y al iniciar aparecera un menu con 5 opciones en pantalla.
Solo se debe de escribir el número de la opción que se desea utilizar y presionar "Enter", yuego, el sistema le irá pidiendo la información necesaria si es que fuera.
adema se puede repetir el proceso todas las veces que se requieran ya que el programa no se cierra automáticamente ya que solo lo hacen en la opción de salir esto para finalizar.

Breve explicación del proyecto: Este programa/proyecto simula cómo se revisan contenidos antes de publicarlos, esto permite tomar decisiones rápidas y también muestra estadísticas de los resultados obtenidos durante su uso precisas.
